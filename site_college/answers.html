2.1 Теоретические вопросы подготовки к лабораторной работе

1. Структура HTML документа

HTML-документ состоит из следующих обязательных элементов:

  • <!DOCTYPE html> — объявление типа документа (HTML5).
  • <html> — корневой элемент документа.
  • <head> — содержит метаданные, такие как заголовок и ссылки на стили.
  • <body> — основной содержательный блок страницы, включает текст, изображения, видео и другие элементы.

2. Новые семантические элементы HTML5

HTML5 ввел несколько новых семантических элементов, которые лучше описывают структуру страницы:

  • <header> — секция для заголовка или навигации.
  • <footer> — секция для футера или контактной информации.
  • <article> — независимая часть контента.
  • <section> — логическая группа контента.
  • <aside> — боковое содержание, не связанное напрямую с основным контентом.
  • <nav> — блок для навигации по сайту.

3. Семантические элементы для работы со структурой страницы

Семантические элементы для структуры страницы в HTML5 включают:

  • <header>
  • <nav>
  • <section>
  • <article>
  • <footer>

4. Регулярные выражения в HTML формах

Регулярные выражения в HTML используются для валидации полей ввода через атрибут pattern. Они позволяют задавать шаблоны для ввода, например, для проверки корректности email или номера телефона.

5. Типы позиционирования элементов

Существует несколько типов позиционирования элементов в CSS:

  • Static — стандартное позиционирование элементов в потоке документа.
  • Relative — относительное позиционирование элемента относительно его исходного местоположения.
  • Absolute — абсолютное позиционирование относительно ближайшего предка с относительным или абсолютным позиционированием.
  • Fixed — фиксированное позиционирование относительно окна браузера.
  • Sticky — комбинация относительного и фиксированного позиционирования, элемент фиксируется при прокрутке.

2.2 Контрольные вопросы

1. Пример структуры HTML5 документа

        <!DOCTYPE html>
        <html lang="ru">
        <head>
            <meta charset="UTF-8">
            <title>Пример HTML5 документа</title>
        </head>
        <body>
            <header>Заголовок сайта</header>
            <nav>Навигация</nav>
            <main>
                <article>Основной контент</article>
            </main>
            <footer>Футер сайта</footer>
        </body>
        </html>

2. Какие новые семантические элементы появились в HTML5?

Новые семантические элементы HTML5:

  • <article>
  • <section>
  • <nav>
  • <aside>
  • <header>
  • <footer>

3. Семантические элементы для текстового содержимого

  • <article> — используется для независимого фрагмента содержимого.
  • <section> — логическая секция документа.
  • <nav> — для секций с навигационными ссылками.
  • <aside> — для боковой информации, не связанной напрямую с основным содержимым.

4. Элементы для поддержки аудио, видео и подключаемых модулей

  • <audio> — элемент для воспроизведения аудиофайлов.
  • <video> — элемент для воспроизведения видеофайлов.
  • <embed> — для встраивания сторонних объектов, например, флеш-анимации.
  • <object> — для встраивания объектов, таких как PDF-файлы.
  • <iframe> — для встраивания сторонних веб-страниц.

5. Типы позиционирования

  • Static — по умолчанию.
  • Relative — позиционирование относительно самого себя.
  • Absolute — абсолютное позиционирование относительно родителя.
  • Fixed — фиксированное относительно окна браузера.
  • Sticky — "липкое" позиционирование.

6. Позиционирование внутри элемента

Внутреннее позиционирование элементов можно контролировать с помощью таких свойств, как text-align для текста или align-items и justify-content для элементов flex-контейнеров.

7. Составные компоненты формы

Основные элементы формы:

  • <input> — поле ввода.
  • <textarea> — многострочное текстовое поле.
  • <select> — выпадающий список.
  • <button> — кнопка.
  • <label> — метка для поля ввода.

8. Глобальные атрибуты HTML5

К глобальным атрибутам в HTML5 относятся:

  • class — задает класс для элемента.
  • id — уникальный идентификатор элемента.
  • style — inline-стили для элемента.
  • data-* — пользовательские атрибуты для хранения данных.
  • title — дополнительная информация, всплывающая при наведении на элемент.
